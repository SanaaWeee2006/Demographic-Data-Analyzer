# Demographic Data Analyzer

This project is part of the freeCodeCamp Data Analysis with Python certification. It uses the Adult Census Income dataset to perform demographic analysis using the Pandas library. 

The program analyzes the dataset and answers the following questions:

* Count of people belonging to each race.
* Average age of men.
Percentage of people with a Bachelor's degree.
Percentage of people with advanced education (Bachelor's, Master's, or Doctorate) who earn more than $50K.
Percentage of people without advanced education who earn more than $50K.
Minimum number of work hours per week.
Percentage of people working the minimum hours who earn more than $50K.
Country with the highest percentage of people earning more than $50K.
Most common occupation among high-income individuals in India.
